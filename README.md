## Simple Chat Application.
#### Tools / Tech used
    * Mocha      (test)
    * React      (view)
    * Redux      (frontend state sync)
    * Socket.IO  (websocket / data transport)
    * Express    (server / HTTP)
    * JSPM       (frontend package management)
    * Node.js

