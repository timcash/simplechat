## Simple Chat Application.
#### Tools / Tech used
    * React      (view)
    * Redux      (state sync)
    * Socket.IO  (websocket)
    * Express    (server / HTTP)
    * JSPM       (frontend package management)
    * Node.js

#### Install
```bash
npm install
jspm install
```
#### Test
```bash
npm test
```

#### Run
```bash
npm start
```
